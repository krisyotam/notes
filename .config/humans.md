
# Index
a list of index files organized by overarching field 

# Jottings 
pre-zettle notes that are not yet validated with sources  

# Lecture
just like marginalia but specific to notes that are from lectures and to be presented in .tex format for 
quickness and ease of use in class or class-like settings. 

# Marginalia 
initial working notes from margins, quick thoughts, ect. validated by sources (.tex, or .md) files that are large and contain 
several notes that are going to be broken up into slipbox notes 

# Slipbox 
the zettlekasten notes 

# Cards 
stores flashcards decks via .csv for easy import into mochi.cards for distilling declarative knowledge 

Jottings --> Marginalia --> Slipbox
