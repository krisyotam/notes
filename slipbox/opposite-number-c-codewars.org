* Problem Statement

:CUSTOM_ID: problem-statement
:END:
#+begin_example
Very simple, given a number (integer / decimal / both depending on the language), find its opposite (additive inverse).

Examples:

1: -1
14: -14
-34: 34
#+end_example

* Chosen Solution
:PROPERTIES:
:CUSTOM_ID: chosen-solution
:END:
Using the additive inverse property we know the following

#+begin_example
In the integers, every element x has a unique element -x such that:
x + (-x) = 0
#+end_example

Now that we know that we can return the inverse of a number by
multiplying it by -x.

#+begin_src C
int opposite(int number) 
{
  return -number; // use additive inverse property 
}
#+end_src
